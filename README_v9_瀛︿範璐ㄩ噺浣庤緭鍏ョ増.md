# 学习质量量化 App v9：低输入高效率版

## 这版目标

这版不是普通打卡软件，而是“记录 + 统计 + 复习 + 错题闭环 + 自动反馈”的学习质量量化系统。

核心原则：

- 能点选就不打字。
- 能自动生成就不手动填。
- 能复用昨天就不重新输入。
- 能批量处理就不一个个处理。
- 每天额外记录时间控制在 3 到 5 分钟以内。

## v9 已加入功能

1. 首页四个大按钮：开始学习、添加错题、待回测、今日反馈。
2. 底部导航：首页、任务、错题、复习、反馈、设置。
3. 快速任务模板：科目、任务类型、数量、时间、难度、策略标签均点选。
4. 结束学习后只点选：专注程度、完成情况、是否产生错题。
5. 深度学习时间自动按专注程度换算。
6. 错题添加流程：拍照、科目、知识点、错因、状态。
7. 错题状态按钮：已订正、已独立重做、完全会、有点忘、不会。
8. 间隔复习：完全会/有点忘/不会自动安排下一次复习。
9. 背诵/默写任务：点选科目、类型、标题、复习节奏。
10. 每日复盘：点选最大问题、主要收获，短句可选。
11. 自动反馈：今日学习质量分、六维雷达图、分项分数、明日建议。
12. 明日建议可以一键加入明日任务。
13. App 类型打勾式归类，包含“学习类应用”。
14. App 类型和全部数据都可以导出 JSON，支持复制和保存。
15. 权限显示模块：缺少使用情况访问权限时会提示。

## 六维雷达图

- 计划执行
- 深度专注
- 主动回忆
- 错题闭环
- 间隔复习
- 身心状态

## 每日学习质量分

- 今日计划：10 分
- 深度专注：15 分
- 有效策略：20 分
- 错题闭环：25 分
- 掌握检测/间隔复习：15 分
- 每日复盘：10 分
- 状态管理：5 分

总分 100 分。

## GitHub 上传方式

解压后打开 `StudentReflect_QualityLowInput_v9` 文件夹。

上传到 GitHub 仓库首页时，必须直接上传这些内容：

```text
.github
app
build.gradle
settings.gradle
README_v9_学习质量低输入版.md
```

不要上传整个 `StudentReflect_QualityLowInput_v9` 文件夹。

正确结构：

```text
仓库首页
├── .github
├── app
├── build.gradle
├── settings.gradle
└── README_v9_学习质量低输入版.md
```

错误结构：

```text
仓库首页
└── StudentReflect_QualityLowInput_v9
    ├── app
    ├── build.gradle
    └── settings.gradle
```

## 打包

上传后进入 GitHub Actions，运行：

```text
Build Android APK v9 Quality Low Input
```

成功后下载：

```text
student-reflect-v9-quality-low-input-apk
```

解压后安装 APK。

## 包名说明

这版包名是：

```text
com.yanjia.studentreflect.v9quality
```

它会作为一个新 App 安装，避免你之前遇到的签名冲突。

旧版数据不能自动进入新版。需要旧版先导出 JSON，再在新版导入。
